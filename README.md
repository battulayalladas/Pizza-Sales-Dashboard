# 🍕 Pizza Sales Analysis | SQL + Excel Dashboard

## Problem Statement
Restaurant had no clear visibility into which pizzas 
sell most, when peak hours occur, and which categories 
drive maximum revenue.

## Tools Used
- SQL — Data extraction & KPI queries
- Microsoft Excel — Dashboard & Visualizations

## Key Findings
- Total Revenue: $8,17,860 | Orders: 21,350
- Peak Days: Friday & Saturday
- Peak Hours: 12-1 PM and 5-8 PM
- Best Seller: Classic Deluxe Pizza (2,453 units)
- Worst Seller: Brie Carre Pizza (490 units)
- Top Category: Classic (26.91% of sales)
- Top Size: Large (45.89% of orders)

## Business Recommendations
- Increase staff on Friday/Saturday evenings
- Consider promoting or removing Brie Carre Pizza
- Focus inventory on Large Classic pizzas
