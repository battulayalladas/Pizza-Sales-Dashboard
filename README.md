# 🍕 Pizza Sales Dashboard

## 📊 Project Overview

This project analyzes pizza sales data to uncover business insights using Power BI, Excel, and SQL.

## 🔧 Tools Used

* Power BI
* Excel
* SQL

## 📈 Key Insights

* Total Revenue: $817K+
* Peak Orders: Weekends (Friday & Saturday evenings)
* Best Seller: Classic Deluxe Pizza
* Lowest Seller: Brie Carre Pizza
* Large pizzas generate maximum revenue

## 📂 Files Included

* Power BI Dashboard (.pbix)
* Excel Dataset (.xlsx)
* SQL Queries (.sql)
* Dashboard Image

## 🔗 Project Purpose

To analyze sales performance, customer behavior, and product trends for better business decisions.
